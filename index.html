1960년, 인터넷 등장

1960년, 핵이 화두
미국이 핵 공격을 당했을 때 어떤 일이 생길 것인지 시뮬레이션을 해봄
당시 중앙집중적이던 통신시스템은 핵공격을 당하면 통신이 마비되는 상황
핵공격에도 견딜 수 있는 통신 시스템으로 인터넷이 탄생
인터넷은 각각의 통신장치들이 전화국 역할을 수행하기에 하나의 통신장치가 사라진다해도 영향이 없음

1990년, 웹 등장

스위스에서 처음 생겨났음
1980년, 스위스 제네바 유럽 입자 물리 연구소에 '팀 버너스 리'가 비정규직으로 취직
프로그래머로 취직하여 'Enquire' 프로그램을 만들었음(지금은 존재하지 않음)
10년동안 연구소를 들어갔다 나왔다하며 'Enquire'와 같은 웹의 전신이 되는 프로그램을 만들었음
1990년, 연구소에 인터넷이 들어옴
10월, 웹페이지를 만드는 편집기를 만들었음
11월, 세계 최초의 웹 브라우저를 만들었음
12월 24일, 팀 버너스 리와 그의 동료는 웹서버라는 프로그램을 만들고, 설치되어있는 컴퓨터에 'info.cern.ch'라는 주소를 부여

인터넷이 동작하는 가장 기본적인 원리

컴퓨터는 최소 두대가 필요함
한쪽은 웹 브라우저, 다른 한쪽은 웹 서버가 깔려있음
웹 서버는 주소를 가지고 있으며, 내부에 파일이 저장되어있음
웹 브라우저에 '주소/파일명'을 검색하면 전기적 신호를 웹 서버가 깔려있는 컴퓨터에 보냄
웹 서버 프로그램이 하드디스크에서 파일을 찾아내 웹 브라우저가 있는 컴퓨터로 보내줌
파일 내부에 있는 코드를 웹 브라우저가 읽어서 웹 동작
정보를 요청하는 쪽은 클라이언트, 응답하는 쪽을 서버라 함
내 컴퓨터의 문서를 볼 수 있게 하려면 웹서버를 직접 깔거나 업체에 맡길 수 있음
